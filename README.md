# pizzeriahub.vercel.app

React + TypeScript front end website developed for a pizza shop

## Tech stack

- React
- TypeScript
- TailwindCSS
- HTML5
- Vercel

## Run project locally

```bash
$ git clone https://github.com/Rha3L/pizzahub.git
$ cd pizzahub
$ npm install
$ npm run dev
```

It requires integrating with external geolocation & pizza data APIs
